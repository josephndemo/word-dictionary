Dictionary Search App
=====================

A simple web-based dictionary application that allows users to search for English word definitions using the Free Dictionary API.

Features
======================

Search for any English word

Displays:

Parts of speech

Definitions

Example sentences (when available)

Clean and simple user interface

Uses a free public dictionary API

No frameworks required

How to Run
======================

Download or clone the project

Open index.html in your web browser

Enter a word in the input box

Click Search to view results

Logo

The application includes a logo displayed at the top of the page for branding.

Notes
======================

An internet connection is required to fetch dictionary data

If a word is not found, an error message will be shown
